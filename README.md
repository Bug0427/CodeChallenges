# Coding Challenges Portfolio

This repository is a curated collection of 20+ progressively challenging coding problems designed to demonstrate practical software engineering skills across multiple languages, paradigms, and problem domains.

Unlike typical algorithm-only practice, these challenges emphasize real-world engineering concerns such as performance, correctness, concurrency, systems behavior, security, and data integrity.

Each challenge is implemented as a self-contained project with clear problem constraints and production-style structure.
